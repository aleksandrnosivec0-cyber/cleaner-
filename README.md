Предварительная настройка (опционально)
По умолчанию пункт [4/11] использует стандартный системный набор очистки. 
Если вы хотите сами выбрать, что именно удалять (например, очищать Корзину или нет), выполните один раз в CMD от админа:
cleanmgr /sageset:1
Выберите нужные пункты и нажмите ОК
после этого настройки сохранятся, при необходимости можно будет поменят снова.
